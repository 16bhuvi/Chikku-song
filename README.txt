
Your Personal Cover Songs Website
---------------------------------

This website is designed to store and stream your COVER SONGS safely.

IMPORTANT:
- Covers cannot legally be offered for download without proper licensing.
- This site provides streaming only (no download button).

HOW TO USE:
1. Inside this folder, create a subfolder named "songs".
2. Put your cover audio files (MP3/WAV) inside the "songs" folder.
   Example: songs/cover1.mp3
3. Open songs.json and edit the entries to match your real song files.

Example songs.json entry:
{
  "title": "Acoustic Cover of XYZ",
  "artist": "Your Name",
  "filename": "xyz_cover.mp3"
}

HOW TO PREVIEW:
Run this command in the folder:
    python -m http.server 8000
Then open:
    http://localhost:8000
